Pattern 4 Monotonic stack — count contribution O(n) · O(n)
Count subarrays where each element is the min/max — multiply left × right spans

Quick decision guide
Exact sum = k, may have negatives?→Pattern 1 — prefix + hashmap
Count subarrays with exactly k distinct?→Pattern 2 — atMost(k) − atMost(k−1)
Non-contiguous subsequences?→Pattern 3 — DP
Count contribution of min/max element?→Pattern 4 — monotonic stack
