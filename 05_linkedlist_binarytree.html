Quick decision guide
Reverse a list range / k-group?→LL-1 — prev/cur/next triple pointer
Remove specific nodes from list?→LL-2 — dummy head + two pointers
Merge, interleave, or sort lists?→LL-3 — find mid, split, merge
Add/remove node preserving BST order?→BT-1 — recurse left/right by value
Reshape or restructure tree?→BT-2 — post-order or Morris traversal
Update values using BST properties?→BT-3 — reverse in-order traversal
