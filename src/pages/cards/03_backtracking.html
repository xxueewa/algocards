LeetCode Patterns

Backtracking

Exhaustive search with pruning — choose, explore, unchoose
Universal template: backtrack(state, choices)
  if isGoal(state): record result & return
  for choice in choices:
    make(choice) → recurse → undo(choice)

Quick decision guide
All subsets / k-size groups?→Pattern 1 — start index, no used[]
All orderings of elements?→Pattern 2 — used[] boolean array
Path in 2D grid?→Pattern 3 — mark '#', restore after DFS
Placement with hard constraints?→Pattern 4 — bitset constraint tracking
Partition or generate valid strings?→Pattern 5 — validate at each cut point
