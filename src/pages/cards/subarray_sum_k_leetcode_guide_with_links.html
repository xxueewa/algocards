Pattern 3 Prefix sum + monotonic deque O(n) time · O(n) space

Use when: array has negatives · sum ≥ k · shortest/longest subarray length

No.ProblemVariantDiff

Build prefix array · maintain increasing deque of indices
For each i: pop front while prefix[i] - prefix[deque.front] ≥ k → candidate answer
Pop back while prefix[i] ≤ prefix[deque.back] (maintain monotone increase)

Quick decision guide

Array has negatives + exact sum = k? → Pattern 1 — prefix + hashmap
Sum divisible by k / modulo condition? → Pattern 1 — track prefix % k in map
All non-negatives + sum ≥ / ≤ k? → Pattern 2 — sliding window
Fixed window of size k? → Pattern 2 — fixed sliding window
Negatives + shortest subarray, sum ≥ k? → Pattern 3 — prefix + deque
Just maximize / minimize subarray sum? → Pattern 4 — Kadane's DP
